## 缓冲区存在的意义
首先就是我理解到的一点：
1. 缓冲区可以起到减少IO次数的作用。因为IO中断程序的运行状态，从用户态转到内核态，这样会拖累程序的运行时间，而缓冲区的存在可以减少IO次数，也就是所谓的<font color="red">优化IO</font>。
